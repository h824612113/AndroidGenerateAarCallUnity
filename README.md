# AndroidGenerateAarCallUnity
安卓生成的aar工程给unity使用，交互。


1.aar包放在unity工程下面的Plugins/Android下面;
2.合并对应的AndroidManifest文件;
3.删除unity中aar包里面的lib下面class.jar包；
4.删除class.jar包里面的Unity的文件夹;
5.打包。
